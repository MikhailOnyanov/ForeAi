Инструкция по работе

Программа позволяет автоматически обрабатывать данные веб-страниц с документацией языка программирования Fore и реализует сопроводительные функции:

– Программа позволяет редактировать набор ссылок на веб-ресурсы для сбора данных

– Выгрузка полученных данных в формате JSON

– Оповещение оператора об аварийных ситуациях и окончании работы программы.

Данное руководство описывает основные элементы управления приложением и возможные ошибки.

Элементы управления

Начать обработку документации
Обрабатывает справку Fore по предоставленному списку сайтов. Если сайт не предоставлен, то обрабатывает тестовые данные.
Импорт файла в векторную базу данных
Импортирует json файл в векторную базу данных. Структура json: [{"Раздел документации": " ", "Текст раздела" : " ", "Версия платформы" : " "},]
Скачать файл результата
Скачивает файл результата обработки документации
Загрузить json файл с ссылками для обработки
Загружает список сайтов для обработки. Структура json: ["",]
